# @myzkeyjs/eslint-config-common

## 0.0.2

### Patch Changes

- Initial release of @myzkeyjs/eslint-config packages
  - **@myzkeyjs/eslint-config-definer**: Type-safe Configurator system with automatic option merging
  - **@myzkeyjs/eslint-config-common**: Base configuration with ignore patterns, linter options, JavaScript rules, and import ordering
  - **@myzkeyjs/eslint-config-typescript**: TypeScript configuration with strict type checking and consistent imports
  - **@myzkeyjs/eslint-config-react**: React configuration with React 17+ support, Hooks rules, and accessibility checks
  - **@myzkeyjs/eslint-config-nextjs**: Next.js configuration supporting both App Router and Pages Router
  - **@myzkeyjs/eslint-config-nodejs**: Node.js/NestJS configuration with server-side rules and decorator support
  - **@myzkeyjs/eslint-config-vitest**: Vitest testing configuration with comprehensive test rules

- Updated dependencies []:
  - @myzkeyjs/eslint-config-definer@0.0.2
